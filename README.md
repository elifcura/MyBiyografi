# 🌐 Elif Cura | Kişisel Portfolyo Web Sitesi

Bu proje, kişisel portfolyo web sitemdir.  
Frontend Developer adayı olarak yeteneklerimi, deneyimlerimi ve projelerimi sergilemek amacıyla geliştirilmiştir.

## Kullanılan Teknolojiler

- HTML5
- CSS3
- Bootstrap 5
- Responsive Design

## Özellikler

- Modern ve responsive tasarım
- Bootstrap grid sistemi
- Karşılama alanı
- Hakkımda bölümü
- Projeler bölümü
- İletişim alanı
- Hover animasyon efektleri
